# turbine-monitoring-rag
A Simple Rag application Chatbot for green house project

1)clone or fork repo 

2)Sign up for a free Streamlit Cloud account (https://streamlit.io/cloud)

3)Connect your GitHub repository to Streamlit Cloud
  Configure your deployment with the following settings:

  Main file path: app.py (or whatever you named your main file)
  
  Python version: 3.9 or later
  
  Add your GOOGLE_API_KEY and OPENAI_API_KEY as a secret in the Streamlit Cloud settings

4)Deploy
